Search Engine

🔹 A.Definisi

1. Web Browser Adalah aplikasi atau softawre untuk membuka dan menampilkan halaman web dari internet. Browser menerjemahkan kode seperti HTML agar dapat dibaca pengguna, dan digunakan untuk membaca artikel, menonton video, serta mengunduh file.

2. Search Engine Adalah layanan untuk mencari informasi di internet menggunakan kata kunci. Mesin pencari seperti Google dan Bing akan menampilkan hasil yang paling relevan dari jutaan halaman web.

3. Variabel Mesin Pencari Merupakan pengaturan atau trik untuk menyaring hasil pencarian, seperti menggunakan tanda kutip untuk mencari frasa tertentu atau “site:” untuk mencari di situs tertentu saja.

4. Pocket Adalah aplikasi yang digunakan untuk menyimpan artikel atau halaman web agar bisa dibaca nanti secara offline. Pocket membantu menyimpan informasi penting agar mudah diakses kapan saja.

🔹 B.Penjelasan

1. Fungsi dan Contoh web browser

Fungsi Utama Web Browser: Menampilkan konten web melalui interpretasi HTML, CSS, dan JavaScript. Memungkinkan navigasi antar halaman web. Fasilitasi pencarian lewat integrasi mesin pencari. Menyimpan halaman favorit (bookmark). Mengunduh file dari internet. Memberikan perlindungan dari ancaman online. Dapat diperluas fungsinya dengan ekstensi dan add-ons.

Contoh Web Browser: Google Chrome, Mozilla Firefox, Microsoft Edg,e Safari, Opera.

2. Search Engine (Mesin Pencari)

Fungsi Mesin Pencari: Menyediakan informasi kepada pengguna. Dan melakukan pencatatan (crawling) dan pengindeksan situs di internet.

Cara Kerja Mesin Pencari: Crawling: Bot mengunjungi dan membaca halaman-halaman web. Indexing: Informasi yang ditemukan disimpan dalam basis data. Ranking: Menyusun hasil pencarian berdasarkan relevansi.

Contoh Search Engine: Google, Bing, Yahoo, Baidu, DuckDuckGo, Yandex.

3. Variabel Mesin Pencari

Jenis-jenis Variabel: Kata Kunci: Kata/kalimat yang dicari. Tanda Kutip (""): Untuk mencari frasa yang tepat. Operator Boolean (AND, OR, NOT): Menggabungkan atau mengecualikan kata kunci. Pencarian Domain (site:): Membatasi ke situs tertentu. Rentang Waktu: Menentukan periode pencarian. Jenis File (filetype:): Mencari format file tertentu seperti PDF, PPT. Lokasi Geografis: Memfokuskan hasil berdasarkan wilayah. Topik Terkait: Menggunakan sinonim atau istilah relevan lainnya. Pertanyaan: Menuliskan pencarian dalam bentuk pertanyaan.

4. Aplikasi Penyimpan Konten Online

Fitur Pocket: Archive: Menyimpan konten yang sudah dibaca atau dihapus. Favorites: Menyimpan konten favorit.

Cara Menggunakan Pocket: Klik ikon Pocket di browser. Tempel atau masukkan URL. Klik “Save”.

🔹 C.Kesimpulan

Web browser dan search engine adalah dua hal berbeda namun saling berkaitan: browser digunakan untuk menjelajah internet, sementara search engine digunakan untuk mencari informasi di internet.

Mesin pencari bekerja melalui proses crawling, indexing, dan ranking agar dapat menyajikan informasi paling relevan kepada pengguna.

Variabel pencarian membantu pengguna mendapatkan hasil yang lebih spesifik dan akurat.

Aplikasi seperti Pocket membantu pengguna menyimpan konten dari internet untuk dibaca kembali kapan saja, bahkan tanpa koneksi internet.
